# Orbit Launcher

Orbit Launcherは、Windows 10/11向けの常駐型ランチャーです。
コマンドパレット、ラジアルメニュー、ボタングリッドから、アプリ・ファイル・フォルダー・URL・定型文・コマンドを素早く呼び出せます。

## ダウンロード

最新版は[GitHub Releases](https://github.com/o1shii/OrbitLauncher-Downloads/releases/latest)からダウンロードしてください。

1. `OrbitLauncher-v0.1.0-win-x64-Setup.exe`をダウンロードします。
2. セットアップを実行します。
3. 画面の案内に従ってインストールします。

インストール後はスタートメニューの「Orbit Launcher」から起動できます。
.NET 8ランタイムを同梱しているため、追加ランタイムのインストールは不要です。

> 現在のセットアップはコード署名されていません。Windowsの警告が表示された場合は、ファイル名とダウンロード元がこのリポジトリであることを確認してください。

## 主な機能

- コマンドパレット
- 5個または8個のラジアルメニュー
- ボタングリッド
- 複数コマンドボードの作成と切り替え
- 使用頻度と直近の使用状況による自動並び替え
- クリップボード履歴
- スクリーンショット履歴
- 一時メモ
- フォルダー内のファイル・ショートカット一括登録
- ブラウザのブックマークHTML一括登録
- Webページのアイコン自動取得
- 操作キーの変更
- Windowsログイン時の自動起動

詳しい操作方法は[使い方](docs/USAGE.md)をご覧ください。

## データ保存

設定、メモ、クリップボード履歴、スクリーンショット履歴は、利用者のPC内に保存されます。

```text
%LOCALAPPDATA%\OrbitLauncher
```

Orbit Launcherには、履歴や設定を外部サーバーへ送信する機能はありません。
Webアイコン取得時は対象サイトへアクセスし、直接取得できない場合はGoogle faviconサービスへサイトのドメイン名のみを送信します。
アプリをアンインストールしても、このフォルダーに保存された設定と履歴は保持されます。

## 動作環境

- Windows 10/11 64-bit
- 追加ランタイム不要
