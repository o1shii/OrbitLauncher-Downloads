# Orbit Launcherの使い方

## 初回起動

初回起動時はコマンドボード管理画面が開きます。
初期コマンドボードにはメモ帳が登録され、使用状況による並び替えとWindowsログイン時の自動起動がONになります。

## 操作キー

- `Ctrl + Space`: コマンドパレットを開く／閉じる
- `Ctrl + Alt + Space`: ラジアルメニューまたはボタングリッドを開く／閉じる
- `Ctrl + Shift + Space`: 次のコマンドボードへ切り替える
- `Esc`: 開いているランチャーを閉じる

操作キーはコマンドボード管理画面から変更できます。変更するキーには`Ctrl`、`Alt`、`Shift`、`Win`のいずれか1つ以上が必要です。

## コマンドの登録

タスクトレイのOrbit Launcherアイコンを右クリックし、「コマンドボードを管理」を開きます。

登録できる操作:

- アプリを起動
- URLを開く
- フォルダーを開く
- ファイルを開く
- テキストをコピー
- コマンドを実行

「一括登録」では、選択したフォルダー直下にあるファイルやショートカットをまとめて登録できます。

「ブックマークを一括登録」では、Chrome、Edge、FirefoxなどからエクスポートしたブックマークHTMLを読み込み、現在のコマンドボードへURLコマンドとして追加できます。入れ子フォルダー内のブックマークも対象になり、同じURLはスキップされます。

URLコマンドの登録時はWebページのアイコンを自動取得します。取得した画像は`%LOCALAPPDATA%\OrbitLauncher\Icons`へ保存され、取得できない場合は表示名の先頭文字が使用されます。

## コマンドボード

コマンドボードは追加、削除、名前変更、並び替えができます。
ボードごとにラジアルメニュー5個、ラジアルメニュー8個、ボタングリッドを選択できます。

「使用状況で並び替え」をONにすると、使用頻度と直近の利用時刻をもとにコマンドが自動整列されます。

## コマンドパレット

コマンドパレットには次のタブがあります。

1. コマンド
2. メモ
3. クリップボード
4. スクリーンショット

メモはパレットを閉じると自動保存されます。クリップボードとスクリーンショットの履歴は最大100件です。

## 履歴の削除

コマンドパレット右上の「履歴を開く」、またはタスクトレイメニューから履歴画面を開けます。
履歴は個別削除または一括消去できます。

## 設定保存先

```text
%LOCALAPPDATA%\OrbitLauncher
```

設定を初期化する場合は、Orbit Launcherを終了してからこのフォルダーを削除してください。
